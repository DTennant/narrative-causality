# 叙事因果律 / Narrative Causality

> 是否存在一种文明，发展到可以太空旅行，但没有整数甚至没有"数"的概念？

一部硬科幻小说的构思与写作空间。

## 核心概念

一种外星存在——"流叙体"——它们的宇宙不由力和质量构成，而由**叙事**构成。事情发生不是因为因果律，而是因为在"故事"中"应该"发生。它们的太空旅行不是把物体从A移到B，而是在两个时空区域之间建立叙事连贯性。

## 主角

天体物理背景的宇航员。母亲是一位一生未成功的小说家。母亲在他执行任务期间去世，留下未完成的手稿。他一直压抑的叙事直觉，最终成为与外星存在沟通的唯一桥梁。
